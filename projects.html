Microsite

Summary

Full screen, single page, fully responsive web application without scrollbars.

Technical Information
  • React javascript library used to build user interface, and runs on Node.js environment.
  • Page content (text, video, and images) are API-consumed from JSON data.
  • Full Screen HTML5 video (muted and looping) as background with semi-transparent white background overlay.
  • The text & slide indicators are horizontally & vertically center-aligned using flexbox layout.
  • Smooth animations for buttons and links.
  • Fade-out and fade-in efects for slide transitions upon clicking the navigation and on mouse-scrolling the page.
  • Background image url changes according to the screen width and orientation.
